C++ 问题汇总

关于C/C++的输入输出精度。

参考文章https://stackoverflow.com/questions/16831623/precision-issue-with-sscanf

  • 输入：
double d;
//here %lf for a double, not %f.
sscanf("5.655035220e-02", "%lf", &d); # it does not lost any precision.

// parse C string str
double atof(const char* str);
double strtod(const char* str, char** endptr);

// parse string
double stod(const stirng& str, size_t* idx=0);
  • 输出：
// format
// %[flags][width][.precision][length]specifier

printf("%*.*g", width, precision, d);

通过调整precision可以得到不同的精度。

详细的格式参数参考http://www.cplusplus.com/reference/cstdio/printf/
